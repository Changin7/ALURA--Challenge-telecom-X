ALURA--Challenge-telecom-X




 Propósito del Análisis
Identificar las causas principales por las que los clientes cancelan sus servicios de telecomunicaciones y encontrar patrones de comportamiento mediante pruebas estadísticas para reducir la tasa de abandono (Churn Rate).

Estructura del Proyecto

* *Datos:* La base de datos (en formato JSON) se consume automáticamente mediante una URL directa desde GitHub. No se requieren archivos locales.

**💡 Insights y Gráficos**
Durante el proyecto se generaron gráficos de pastel, mapas de calor (correlaciones), y gráficos de densidad/barras que revelan:
* **Costos:** Los clientes que abandonan el servicio pagan cargos mensuales significativamente más altos.
* **Fidelidad:** A menor antigüedad del cliente, mayor es la probabilidad de cancelación.
* **Contratos:** Los clientes con contratos "mes a mes" (Month-to-Month) representan la mayor tasa de abandono.
